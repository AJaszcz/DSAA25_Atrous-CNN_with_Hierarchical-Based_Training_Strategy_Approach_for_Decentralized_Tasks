## Overview

This repository contains the source code related to the paper:

**_Atrous-CNN with Hierarchical-Based Training Strategy Approach for Decentralized Tasks_**

to be presented at the **DSAA 2025 (IEEE International Conference on Data Science and Advanced Analytics)**.

The proposed framework is based on ACBAM CNN architecture with hierarchical learning strategy for federated systems.
